Conexiones entre Usuario y Netflix

Relaciones y flujos de interacción en el sistema

1
Usuario → Iniciar Sesión
El usuario se autentica en el sistema proporcionando sus credenciales (email y contraseña).
→
Usuario a Sistema
2
Usuario → Reproducir Contenido
El usuario selecciona y reproduce una película, serie o documental.
→
Usuario a Sistema
3
Usuario → Buscar Contenido
El usuario utiliza la función de búsqueda para encontrar títulos específicos.
→
Usuario a Sistema
4
Usuario → Gestionar Mi Lista
El usuario agrega o elimina contenido de su lista personal.
→
Usuario a Sistema
5
Sistema → Recomendar Contenido
Netflix sugiere contenido basado en el historial de visualización y preferencias.
→
Sistema a Usuario
6
Usuario → Gestionar Perfiles
El usuario crea, modifica o elimina perfiles en su cuenta.
→
Usuario a Sistema
7
Sistema → Adaptar Calidad de Video
Netflix ajusta automáticamente la calidad del streaming según la velocidad de conexión.
→
Sistema a Usuario
8
Usuario → Configurar Cuenta
El usuario modifica la configuración de su cuenta y preferencias.
→
Usuario a Sistema
